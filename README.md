# Installation instructions
* Download the contents of this repo into a local folder, say, TabGrouperExtension.
* Go to `chrome://extensions/` in your Chrome browser.
* On the top right, enable `Developer mode`.
* Click on `Load unpacked` on the top left.
* Select the folder where you saved the contents of this repo.
